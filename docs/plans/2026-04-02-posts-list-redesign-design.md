# Posts List Redesign Design

**Goal:** 把文章列表页从“带筛选的工具页”调整为更有博客栏目气质的内容入口页。

**Context:**
- 当前页头以搜索表单为中心，栏目感弱。
- 文章卡片面积偏大、信息层次偏少，内容入口感不足。
- 页面底部在文章较少时收尾过空。

**Design Direction:**
- 页头改为栏目式介绍区，保留搜索与标签筛选，但弱化表单感，强化“文章栏目”定位。
- 列表建立主次关系：首篇文章作为主入口，其余文章更紧凑，并增加元信息层次。
- 在列表底部加入轻量收尾区，让分页与继续浏览更自然。

**Key Decisions:**
- 保持现有数据流与查询参数逻辑，不改 API 交互方式。
- 优先复用现有 `PostCard`，通过增加 `featured` 变体降低结构复杂度。
- 风格延续首页的个人博客方向，避免商业官网感和过重卡片感。
